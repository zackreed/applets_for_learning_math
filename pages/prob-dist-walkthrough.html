üéØ Introduction

A probability distribution describes how likely different outcomes are for a random variable. For continuous random variables, we use a probability density function (PDF) to represent the distribution. The area under the PDF curve equals 1, representing 100% probability.

Key Concept: Probability Density Functions

A probability density function \(f(x)\) satisfies:

  • \(f(x) \geq 0\) for all \(x\) (non-negative)
  • \(\int_{-\infty}^{\infty} f(x) \, dx = 1\) (total probability is 1)
  • \(P(a \leq X \leq b) = \int_a^b f(x) \, dx\) (probability of interval)

Different distributions model different types of random phenomena!

üìö Distribution Catalog

1. Normal (Gaussian) Distribution

PDF: \(f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{(x-\mu)^2}{2\sigma^2}}\)

Parameters: \(\mu\) (mean), \(\sigma\) (standard deviation)

Properties:

  • Bell-shaped, symmetric around the mean
  • 68% of data within \(\pm 1\sigma\), 95% within \(\pm 2\sigma\), 99.7% within \(\pm 3\sigma\)
  • Models many natural phenomena (heights, measurement errors, etc.)
  • Mean = Median = Mode = \(\mu\)

2. Exponential Distribution

PDF: \(f(x) = \lambda e^{-\lambda x}\) for \(x \geq 0\)

Parameters: \(\lambda\) (rate parameter)

Properties:

  • Models time between events in a Poisson process
  • Memoryless property: past doesn't affect future
  • Mean = \(1/\lambda\), Variance = \(1/\lambda^2\)
  • Used for: lifetimes, waiting times, radioactive decay

3. Uniform Distribution

PDF: \(f(x) = \frac{1}{b-a}\) for \(a \leq x \leq b\), else 0

Parameters: \(a\) (minimum), \(b\) (maximum)

Properties:

  • All values in range \([a, b]\) are equally likely
  • Rectangular/flat distribution
  • Mean = \((a+b)/2\), Variance = \((b-a)^2/12\)
  • Used for: random number generation, lack of information scenarios

4. Gamma Distribution

PDF: \(f(x) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}\) for \(x > 0\)

Parameters: \(\alpha\) (shape), \(\beta\) (rate)

Properties:

  • Generalizes exponential distribution (exponential is Gamma with \(\alpha=1\))
  • Models sum of \(\alpha\) independent exponential random variables
  • Mean = \(\alpha/\beta\), Variance = \(\alpha/\beta^2\)
  • Used for: insurance claims, rainfall amounts, waiting times

5. Cauchy Distribution

PDF: \(f(x) = \frac{1}{\pi\gamma\left[1 + \left(\frac{x-x_0}{\gamma}\right)^2\right]}\)

Parameters: \(x_0\) (location), \(\gamma\) (scale)

Properties:

  • Heavy tails - extreme values are more likely than in normal distribution
  • Mean and variance are undefined (do not exist!)
  • Median = Mode = \(x_0\)
  • Used in physics (resonance), financial modeling
  • Example of how not all distributions have finite moments

6. Laplace Distribution

PDF: \(f(x) = \frac{1}{2b} e^{-\frac{|x-\mu|}{b}}\)

Parameters: \(\mu\) (location), \(b\) (scale)

Properties:

  • Sharp peak at \(\mu\) with exponential tails
  • More peaked than normal, heavier tails than normal
  • Mean = Median = Mode = \(\mu\), Variance = \(2b^2\)
  • Used in: signal processing, machine learning (Lasso regression)

üîó Connection to Statistics & Calculus

Expected Value (Mean)

For a continuous random variable with PDF \(f(x)\), the expected value is:

\(\mu = E[X] = \int_{-\infty}^{\infty} x \cdot f(x) \, dx\)

This integral may not converge (as with the Cauchy distribution), in which case the mean is undefined.

Variance and Standard Deviation

The variance measures the spread of the distribution:

\(\sigma^2 = \text{Var}(X) = E[(X-\mu)^2] = \int_{-\infty}^{\infty} (x-\mu)^2 \cdot f(x) \, dx\)

The standard deviation is \(\sigma = \sqrt{\sigma^2}\).

Probability as Area

The probability that a random variable falls in an interval \([a, b]\) is the area under the PDF curve:

\(P(a \leq X \leq b) = \int_a^b f(x) \, dx\)

This is a direct application of the definite integral from calculus!

Central Limit Theorem

One of the most important theorems in statistics: regardless of the underlying distribution, the sum (or average) of many independent random variables tends toward a normal distribution as the number of variables increases. This explains why the normal distribution appears so frequently in nature!

Applications by Distribution

  • Normal: IQ scores, heights, measurement errors, natural phenomena
  • Exponential: Time between arrivals, radioactive decay, component lifetimes
  • Uniform: Random number generation, prior distributions in Bayesian statistics
  • Gamma: Insurance claims, rainfall, queue waiting times
  • Cauchy: Resonance in physics, ratio of two normal random variables
  • Laplace: Financial returns, signal processing, machine learning regularization
