✅ Multivariable Chain Rule Applet - Test Page

Test Instructions:

  1. The applet should load in the iframe below
  2. Try dragging to rotate the view
  3. Scroll to zoom in/out
  4. Click on the domain plane to set a point
  5. Use the control panel to:
    • Change surface functions
    • Adjust direction angle and magnitude
    • Toggle display elements
  6. Verify that chain rule values update correctly

Embedded Applet:

Expected Behavior:

  • Visual Elements:
    • Green 3D surface with lighting
    • Grid on the xy-plane
    • Red point on surface
    • Yellow curve through surface
    • Yellow tangent plane
    • Orange partial derivative vectors
    • Cyan total derivative vector
  • Interactions:
    • Smooth rotation when dragging
    • Zoom responds to scroll wheel
    • Point placement updates when clicking
    • Sliders update direction vector in real-time
  • Mathematical Accuracy:
    • dz/dt = (∂z/∂x)(dx/dt) + (∂z/∂y)(dy/dt)
    • Values should change as you move the point
    • Direction changes should affect dz/dt

Test Cases:

  1. Paraboloid at origin:
    • Set point at (0, 0)
    • Both partial derivatives should be 0
    • dz/dt should be 0 regardless of direction
  2. Paraboloid at (1, 0):
    • ∂z/∂x should be 2
    • ∂z/∂y should be 0
    • If moving in x-direction (angle ≈ 1.57), dz/dt ≈ 2
  3. Saddle at origin:
    • Both partial derivatives should be 0
    • Surface should curve up in x-direction, down in y-direction
  4. Toggle features:
    • Grid toggle should hide/show grid lines
    • Curve toggle should hide/show yellow curve
    • Tangent Plane toggle should hide/show yellow plane
    • Vectors toggle should hide/show all arrows

✅ If all tests pass, the applet is ready to use!
