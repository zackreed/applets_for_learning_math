Gradient Descent Visualization - Standalone JS Example

üìö How to Use the Standalone JavaScript

This page demonstrates the GradientDescentViz class, which can be used in any JavaScript environment.

Basic Usage:

// 1. Include the script
<script src="path/to/gradient-descent-standalone.js"></script>

// 2. Create a container element
<div id="viz-container"></div>

// 3. Initialize the visualization
const container = document.getElementById('viz-container');
const viz = new GradientDescentViz(container, {
  width: 1300,
  height: 750,
  learningRate: 0.01,
  numDataPoints: 20,
  showControls: true // Set to false for embedded use
});

Public API Methods:

// Set learning rate
viz.setLearningRate(0.05);

// Take gradient descent steps
viz.step(); // Take 1 step
viz.step(10); // Take 10 steps

// Reset to new random position
viz.reset();

// Generate new data
viz.newData();

// Get current state
const state = viz.getState();
console.log('m0:', state.m0, 'm1:', state.m1, 'error:', state.error);
